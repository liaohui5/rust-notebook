## 环境安装

- [去官网下载,手动安装](https://www.rust-lang.org/tools/install)
- [外网下载比较慢,可以使用 rsproxy](https://rsproxy.cn/#getStarted) 推荐使用这种方式

## hello world

1. 新建 `main.rs`
2. 输入如下代码

```rs
fn main() {
  println!("hello world");
}
```

3. 编译 & 运行

```sh
# 编译
rustc ./main.rs

# 执行
./main # ./main.exe (on windows)
```
